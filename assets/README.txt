Drop your images in this folder. The site looks for these filenames
(they match your current live site, so you can download them from
https://alexpaces.com/assets/ and place them here):

  ProfilePic.jpg              — hero portrait
  junepoint-preview2.PNG      — JunePoint project preview
  jpa-preview2.jpg            — JPA Contractors preview
  ico-emergency-preview.PNG   — ICO Emergency preview
  clicker-preview.PNG         — Epic Clicker preview
  pong-preview.PNG            — Pong Classic preview
  Cert.jpg                    — certification image

If a file is missing here, the site automatically falls back to the
copy hosted on alexpaces.com. To change filenames, edit the "image"
fields in data/portfolio.json.
